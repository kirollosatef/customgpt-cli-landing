Quick Start

Up and running in 60 seconds

1

Install

Install the CLI globally from npm. Works on macOS, Linux, and Windows.

$ npm install -g @customgptai/cli
2

Authenticate

Log in with your CustomGPT.ai account. Browser-based OAuth opens automatically.

$ customgptai auth login
3

Build

Start managing your AI agents, conversations, and data sources instantly.

$ customgptai agents list
Features

Everything you need, nothing you don't

Complete API Coverage

75 commands across 17 topics. Every endpoint is a CLI command — auto-generated from the OpenAPI spec.

Real-Time Streaming

SSE streaming for conversations. Watch responses arrive token by token, just like the web interface.

Secure Authentication

Browser-based OAuth with CSRF protection, or set an API key directly. Tokens stored locally.

Flexible Output

JSON, CSV, table, or machine-readable. Field selection, piping support, and scriptable output.

Built-in Resilience

Automatic retries with exponential backoff. Rate limit headers respected. Configurable retry count.

Zero Configuration

Works out of the box. Install, authenticate, go. Customize defaults when you need to.
